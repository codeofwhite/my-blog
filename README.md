# MyBlog-CodeMist - 个人博客系统

重庆大学 2022 级软件综合实践课程项目 | 基于 Vue3 + Node.js 全栈个人博客系统

## 项目简介
本项目是重庆大学大二软件综合实践课程作业，是一套前后端分离的完整个人博客系统。包含前台博客展示、后台管理系统、Node 服务端三层架构，支持文章发布、编辑、删除、点赞、用户权限管理等功能。

## 核心功能
### 前台页面
- 博客文章展示。
- 点赞功能。
- 响应式设计。

### 后台管理
- 文章管理（发布、编辑、删除）。
- 用户管理（权限、角色）。

### 服务端
- 提供博客文章、用户和点赞等数据的 API 接口。
- 数据库交互（如 MongoDB 或 MySQL）。

## 项目分支结构
- master：前台用户界面
- admin：后台管理界面
- back：Node.js 后端服务

## 快速启动
1. 启动前端（当前分支）
```bash
运行
# 安装依赖
npm install
```
```
# 开发模式启动
npm run dev
```
```
# 构建生产版本
npm run build
```
2. 启动后端（back 分支）
进入 back 分支目录
```
bash
运行
npm install
npm run start
```
3. 启动管理后台（admin 分支）
进入 admin 分支目录
```
bash
运行
npm install
npm run dev
```
